안녕하세요! Claude Code 잘 쓰고 계시죠? 👋

혹시 *"나는 AI를 잘 쓰고 있는 걸까?"* 생각해본 적 있으신가요?

이걸 자동으로 측정해주는 플러그인을 만들었습니다.

*🔍 AX-Eval — 내 AI 활용 수준 자동 측정*

Claude Code 대화 로그를 분석해서 4가지 축으로 활용 수준을 알려줘요.
• *요청력* — AI에게 배경·목적·형식을 얼마나 잘 전달하는가
• *검증력* — AI 결과를 그냥 쓰지 않고 확인하고 수정하는가
• *활용력* — Claude Code의 다양한 기능을 쓰는가
• *판단력* — AI를 언제, 어떻게 쓸지 전략적으로 판단하는가

결과는 ⭐~⭐⭐⭐⭐⭐ 별점으로 나와서 직관적이에요!

*🤖 자동 요청 코치 기능*
설치하면 Claude가 모호한 요청을 감지해서 더 나은 방법을 알려줘요.

```
나: 지난달 이벤트 성과 정리해줘

Claude: (정리 결과...)

💡 요청 팁: "3월 앱푸시 이벤트 성과를 발송수/오픈율/전환율 기준으로
   주차별 표로 정리해줘" 처럼 대상·지표·형식을 같이 말하면
   한 번에 원하는 결과가 나와요.
```

요청이 충분히 구체적이면 팁은 자동으로 생략됩니다.
*어떤 프로젝트에서 Claude Code를 열든 자동으로 작동해요.* (설치 1회로 끝)

*📦 설치 (Claude Code 터미널, 순서대로)*
```
claude plugins marketplace add pms0505-lgtm/biz-plugins
claude plugins install ax-eval@biz-plugins
```
> ⚠️ 설치 후 Claude Code를 껐다 다시 켜야 적용돼요!

*💬 명령어 3개가 끝*
```
/ax-eval 시작   → 처음 한 번만 (역할 선택, 1분)
/ax-eval 체크   → 내 레벨 확인 (주 1회 권장)
/ax-eval 팁     → 약한 부분 맞춤 성장 팁
```

> • Claude Code 대화 3번 이상 있어야 분석 가능해요
> • 분석은 내 PC 안에서만 이루어져요 (외부 전송 없음)
> • 설치 후에는 세션 시작할 때 약한 축을 자동으로 알려주는 기능도 있어요
